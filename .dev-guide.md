# Amanwal Development

Archivo para fácil desarrollo en ambos entornos.

## Iniciar Ambos Servicios

### Opción 1: Comandos separados (recomendado)

Terminal 1 - Backend:
```bash
cd backend
npm run dev
```

Terminal 2 - Frontend:
```bash
cd frontend
npm run dev
```

### Opción 2: Con npm workspaces (requiere terminal multiplex)

```bash
npm run dev
```

## Instalar Todas las Dependencias

```bash
npm run install-all
```

O manualmente:
```bash
npm install
cd backend && npm install
cd ../frontend && npm install
```

## Setup Base de Datos

```bash
npm run db:setup
```

## Build para Producción

```bash
npm run build
```

## Testing

```bash
npm run test
```

---

✨ La aplicación estará disponible en `http://localhost:5173`
